## 界面总览

交易助手界面采用垂直布局设计，从上到下依次排列各个功能区域，确保操作流程的逻辑性和便利性。

## 标题栏区域

### 信息显示
- **货币对名称**: 当前选择的交易货币对
- **点差显示**: 实时点差数值，带有点差图标
- **价格变化**: 当日价格变化百分比，正值绿色，负值红色

### 视觉特性
- **背景色**: 深灰色背景，确保信息清晰可读
- **居中对齐**: 货币对名称居中显示
- **图标标识**: 点差和价格变化都有对应图标

### 手数调整区域

位于两个主交易按钮中间，包含：

#### 手数输入框
- **位置**: 中央位置
- **宽度**: 95像素
- **对齐**: 右对齐显示
- **精度**: 显示两位小数

#### 调整按钮
<Accordion title="粗调按钮（左侧）">
- **+按钮**: 增加0.1手数
- **-按钮**: 减少0.1手数
- **位置**: 手数框左侧
- **用途**: 快速大幅调整手数
</Accordion>

<Accordion title="精调按钮（右侧）">
- **+按钮**: 增加0.01手数
- **-按钮**: 减少0.01手数
- **位置**: 手数框右侧
- **用途**: 精确微调手数
</Accordion>

## 价格显示区域

### BID价格（卖价）
显示在SELL按钮上，价格分段显示：

#### 价格分段规则
- **整数部分**: 使用Arial Black字体，标准大小
- **小数主要部分**: 使用Arial字体，加大15号显示
- **小数次要部分**: 使用Arial Black字体，标准大小

#### 显示逻辑
```
对于3位及以上小数的货币对：
BID: 1.2345 → "1.23" + "45" + ""

对于2位小数的货币对：
BID: 110.25 → "110." + "25"
```

### ASK价格（买价）
显示在BUY按钮上，分段规则与BID价格相同。

### 价格特性
- **居中对齐**: 价格在按钮中居中显示
- **颜色适配**: 根据市场状态调整文字颜色
- **实时更新**: 价格变化时自动刷新

## 挂单按钮区域

### 按钮布局
采用2x2网格布局：

| 左列 | 右列 |
|------|------|
| Sell Limit | Buy Stop |
| Sell Stop | Buy Limit |

### 按钮特性
- **尺寸**: 每个按钮141x30像素
- **颜色**: 卖出按钮红色系，买入按钮蓝色系
- **状态**: 支持按下效果和禁用状态
- **文字**: 白色文字，清晰可读

## 网格配置区域

### 计算方法选择

<CardGroup cols={2}>
  <Card title="Martin（马丁格尔）" icon="chart-line-up">
    - **策略**: 亏损后加倍手数
    - **公式**: volume = base × multiple^i
    - **默认倍率**: 1.5
    - **风险**: 较高，需要充足资金
  </Card>
  <Card title="Stack（等量加仓）" icon="layers">
    - **策略**: 保持相同手数递增
    - **公式**: volume = base + i × increment
    - **默认增量**: 0.01
    - **风险**: 较低，资金需求稳定
  </Card>
</CardGroup>

### 倍率/增量设置
- **马丁格尔模式**: 设置倍数（如1.5表示每次增加50%）
- **等量加仓模式**: 设置增量（如0.01表示每次增加0.01手）

## 货币对选择区域

### 下拉选择框
- **功能**: 选择要交易的货币对
- **更新**: 选择后自动更新报价和参数
- **支持**: 所有经纪商提供的交易品种

### 订单设置
- **订单备注**: 自定义订单注释，默认"Copilot"
- **魔术号**: 订单标识号，默认2025

## 界面交互特性

### 拖拽功能
- **标题栏拖拽**: 点击标题栏可拖拽整个面板
- **位置记忆**: 面板位置会被记住
- **边界限制**: 不会拖拽到屏幕外

### 响应式设计
- **自适应**: 根据屏幕DPI自动调整字体大小
- **多语言**: 支持中英文界面切换
- **主题适配**: 根据MT5主题调整颜色

### 状态反馈
- **按钮状态**: 按下、悬停、禁用等状态
- **颜色变化**: 根据市场状态和操作结果变化
- **声音提示**: 重要操作有声音反馈

## 使用技巧

<Tip>
**界面优化建议**:
- 将面板放置在图表左侧或右侧，避免遮挡价格
- 根据个人习惯调整面板位置
- 熟悉各按钮位置，提高操作效率
- 注意观察按钮状态变化，确认操作有效性
</Tip>
