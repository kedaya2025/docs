## 模块简介

历史统计模块是 TraderCopilot 的专业交易分析工具，提供全面的历史交易数据统计和分析功能。该模块能够从多个维度分析交易表现，生成详细的统计报告和可视化图表，帮助交易者深入了解交易策略的效果和改进方向。

### 全面统计指标

系统提供14项核心统计指标：

#### 基础指标

<CardGroup cols={2}>
  <Card title="交易量指标" icon="scale-balanced">
    **订单数量**: 统计期间的总订单数

    **总手数**: 累计交易手数

    **最大手数**: 单笔最大交易量

    **最小手数**: 单笔最小交易量
  </Card>
  <Card title="成本指标" icon="dollar-sign">
    **隔夜费**: 持仓过夜产生的费用

    **手续费**: 交易佣金和点差成本

    **总成本**: 所有交易相关费用
  </Card>
</CardGroup>

#### 收益指标

<CardGroup cols={2}>
  <Card title="利润分析" icon="trending-up">
    **总利润**: 净利润金额

    **利润率**: 相对于投入资金的收益率

    **毛利润**: 所有盈利交易的总和

    **毛亏损**: 所有亏损交易的总和
  </Card>
  <Card title="资金管理" icon="wallet">
    **出入金**: 资金流入流出记录

    **余额变化**: 账户余额历史变化

    **净值变化**: 账户净值波动情况
  </Card>
</CardGroup>

#### 风险指标

<CardGroup cols={2}>
  <Card title="回撤分析" icon="chart-line-down">
    **最大回撤**: 最大亏损金额

    **回撤百分比**: 相对于峰值的回撤比例

    **回撤持续时间**: 回撤恢复所需时间
  </Card>
  <Card title="浮盈浮亏" icon="chart-mixed">
    **浮盈金额**: 未平仓盈利

    **浮盈百分比**: 浮盈占总资金比例

    **浮亏风险**: 未平仓亏损风险
  </Card>
</CardGroup>

#### 效率指标

<CardGroup cols={2}>
  <Card title="成功率" icon="percentage">
    **胜率**: 盈利交易占总交易的比例

    **盈亏比**: 平均盈利与平均亏损的比值

    **期望收益**: 每笔交易的期望盈利
  </Card>
  <Card title="时间效率" icon="clock">
    **最大持仓时间**: 单笔交易最长持有时间

    **平均持仓时间**: 所有交易的平均持有时间

    **交易频率**: 单位时间内的交易次数
  </Card>
</CardGroup>

## 筛选功能

### 三种筛选模式

<Accordion title="默认模式">
  **功能**: 统计所有历史交易数据

  **适用**: 全面了解整体交易表现

  **优势**: 数据完整，分析全面

  **使用**: 初次分析或整体评估时使用
</Accordion>

<Accordion title="仅统计选定元素">
  **功能**: 只统计选中的魔术号或货币对

  **适用**: 分析特定策略或品种的表现

  **优势**: 精确定位，针对性强

  **使用**: 策略优化或品种分析时使用
</Accordion>

<Accordion title="不统计选定元素">
  **功能**: 排除选中的魔术号或货币对

  **适用**: 排除异常数据或特殊情况

  **优势**: 过滤干扰，突出主要数据

  **使用**: 排除测试数据或异常交易时使用
</Accordion>

## 界面布局

历史统计面板采用紧凑的垂直布局：

### 主要区域

1. **标题栏** - 显示模块名称和账户信息
2. **维度选择** - 时间维度和筛选维度切换
3. **时间控制** - 日期选择和时间范围设置
4. **图表区域** - 双图表并排显示
5. **统计表格** - 分页显示详细统计数据
6. **分页控制** - 导航和页面跳转功能

### 设置面板

1. **筛选方法** - 三种筛选模式选择
2. **元素管理** - 魔术号和货币对列表管理
3. **样式设置** - 字体、颜色、显示行数配置

<Accordion title="策略评估">

  通过回撤和浮盈浮亏分析，评估和改进风险控制措施

  生成专业的交易绩效报告，用于投资者汇报或自我评估

## 动态列切换

系统支持点击列标题切换显示内容：

| 列名       | 切换选项              | 说明      |
| -------- | ----------------- | ------- |
| **手数**   | 总手数 → 最大手数 → 最小手数 | 分析交易规模  |
| **费用**   | 隔夜费 ↔ 手续费         | 分析成本构成  |
| **利润**   | 利润金额 ↔ 利润百分比      | 绝对值与相对值 |
| **回撤**   | 回撤金额 ↔ 回撤百分比      | 风险评估    |
| **浮盈**   | 浮盈金额 ↔ 浮盈百分比      | 未实现收益   |
| **毛利**   | 毛利润 ↔ 毛亏损         | 盈亏分析    |
| **持仓时间** | 最大时间 ↔ 平均时间       | 时间效率    |

## 注意事项

<Warning>
  **数据准确性**

  - 统计数据基于MT5的历史交易记录
  - 确保历史数据完整性，避免数据缺失
  - 注意时区设置对时间统计的影响
  - 定期备份历史数据防止丢失
</Warning>

<Info>
  **使用建议**

  - 建议定期进行历史统计分析
  - 结合不同维度的统计结果进行综合分析
  - 关注趋势变化而非单一时点数据
  - 将统计结果用于策略优化和风险控制
</Info>
