## 模块概览

TraderCopilot 包含七个核心功能模块，每个模块都专注于交易流程中的特定环节，共同构成完整的交易生态系统。

<CardGroup cols={2}>
  <Card title="历史统计" icon="chart-bar" href="/modules/history">
    **核心功能**: 交易记录管理、统计报告、数据导出

    **适用场景**: 数据管理、绩效统计、合规报告
  </Card>
  <Card title="账户分析" icon="chart-line" iconType="regular" href="/modules/Analyse">
    **核心功能**: 交易历史分析、数据可视化、报告生成

    **适用场景**: 交易复盘、策略评估、绩效分析
  </Card>
  <Card title="财经新闻" icon="newspaper" href="/modules/NewsPanel">
    **核心功能**: 实时新闻、事件交易、时区管理

    **适用场景**: 基本面分析、新闻交易、事件驱动策略
  </Card>
  <Card title="风险控制" icon="shield-check" href="/modules/RiskControl">
    **核心功能**: 风险监控、资金管理、预警系统

    **适用场景**: 风险防控、资金保护、合规管理
  </Card>
  <Card title="交易助手" icon="hand-holding-dollar" href="/modules/Assistant">
    **核心功能**: 交易操作面板、订单管理、快速下单

    **适用场景**: 日常交易操作、订单管理、风险控制
  </Card>
  <Card title="市场动态" icon="chart-mixed" href="/modules/MarketWatch">
    **核心功能**: 市场情绪监控、趋势分析、自动交易

    **适用场景**: 市场研判、入场时机选择、情绪交易
  </Card>
  <Card title="快速跟单" icon="link" href="/modules/CopyTrading">
    **核心功能**: 多模式跟单、跨平台、网络跟单

    **适用场景**: 本地多账户复制交易、跨区域跟随
  </Card>
  <Card title="篮子交易" icon="basket-shopping" href="/modules/basket">
    **核心功能**: 货币强度分析、组合交易、风险分散

    **适用场景**: 分散投资、套利交易、货币对冲
  </Card>
  <Card title="更多功能" icon="chart-bar">

  </Card>
</CardGroup>

## 使用建议

### 新手用户学习路径

<Steps>
  <Step title="交易助手">
    从基础的交易操作开始，熟悉下单、平仓等基本功能
  </Step>
  <Step title="风险控制">
    学习风险管理的重要性，设置合适的风险参数
  </Step>
  <Step title="市场分析">
    掌握市场分析方法，了解市场情绪和趋势判断
  </Step>
  <Step title="分析模块">
    学会分析交易表现，持续改进交易策略
  </Step>
</Steps>

## 模块特性对比

| 模块       | 主要功能      | 自动化程度 | 复杂度 | 推荐用户 |
| -------- | --------- | ----- | --- | ---- |
| **分析模块** | 数据分析、图表展示 | 中等    | 简单  | 所有用户 |
| **交易助手** | 手动交易、订单管理 | 低     | 简单  | 所有用户 |
| **市场分析** | 情绪分析、自动交易 | 高     | 中等  | 进阶用户 |
| **新闻面板** | 新闻交易、事件驱动 | 高     | 复杂  | 专业用户 |
| **篮子交易** | 组合策略、货币强度 | 高     | 复杂  | 专业用户 |
| **历史统计** | 数据管理、报告生成 | 中等    | 中等  | 所有用户 |
| **风险控制** | 风险监控、资金管理 | 中等    | 中等  | 所有用户 |

## 模块间协作

TraderCopilot 的各个模块可以只有搭配相互协作：

<Tip>
  **数据流向**: 交易助手 → 历史统计 → 分析模块 → 策略优化

  **决策支持**: 市场分析 \+ 新闻面板 → 交易决策 → 风险控制

  **自动化链条**: 市场情绪/新闻事件 → 自动交易 → 风险监控 → 止盈止损
</Tip>
